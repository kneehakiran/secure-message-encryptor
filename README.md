# 🔐 Secure Message Encryptor

A beginner-friendly GUI-based message encryption tool built using Python and Tkinter. Encrypt and decrypt text messages using a secret numeric key — all within a clean dark-themed desktop interface.

---

## 🚀 Features

- 🔒 Encrypt any text message using a secret numeric key
- 🔓 Decrypt encrypted messages back to original text
- 🖥️ Interactive dark-themed desktop GUI
- ⚡ Real-time output display
- 🛡️ Input validation with error handling
- 🧠 Built on Caesar Cipher encryption logic

## ▶️ How to Run

1. Make sure Python 3 is installed on your system
2. Clone this repository:
git clone https://github.com/kneehakiran/secure-message-encryptor.git
3. Navigate to the folder:
cd secure-message-encryptor
4. Run the file:
python file_encryption_tool_gui.py
> No external libraries needed — Tkinter comes built-in with Python.
