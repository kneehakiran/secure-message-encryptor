# 🔐 File Encryption Tool (Python + Tkinter)

A beginner-friendly GUI-based File Encryption Tool built using Python and Tkinter.
This application allows users to encrypt and decrypt messages using a secret key with a clean dark-themed desktop interface.

---

## 🚀 Features

✅ Encrypt Messages
✅ Decrypt Messages
✅ Secret Key Based Encryption
✅ Interactive Tkinter GUI
✅ Dark-Themed Desktop Interface
✅ Real-Time Output Display
